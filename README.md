### 👨‍💻 Обо мне (About Me)
Я независимый разработчик, увлекающийся системными утилитами, кастомизацией интерфейсов и автоматизацией на Python и C/C++. Люблю переносить крутые Unix-инструменты в экосистему Windows и делать их доступными для каждого.

- 🚀 **Главный проект сейчас:** [PMatrix](https://github.com/HeyImdaycreate/PMatrix) — полноценный, кастомизируемый клон легендарного `cmatrix` для Windows.
