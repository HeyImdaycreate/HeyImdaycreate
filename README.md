# Привет! Я HeyImdaycreate 👋 

### 👨‍💻 Обо мне (About Me)
Я независимый разработчик, увлекающийся системными утилитами, кастомизацией интерфейсов и автоматизацией на Python и C/C++. Люблю переносить крутые Unix-инструменты в экосистему Windows и делать их доступными для каждого.

- 🛠️ **Текущий фокус:** Оптимизация консольных анимаций и разработка CLI-инструментов.
- 🚀 **Главный проект сейчас:** [PMatrix](https://github.com) — полноценный, кастомизируемый клон легендарного `cmatrix` для Windows.
- ⚡ **Девиз:** "Если утилита не работает на моей OS, я просто перепишу её сам".
